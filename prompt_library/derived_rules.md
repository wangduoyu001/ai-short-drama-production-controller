# derived_rules 规则提炼版

本文件只保存从上传的 1人漫剧提示词库中提炼出的通用生产规则，不复制第三方模板原文。

## extracted_value 提炼价值

- 忠实原文、对白零篡改、全量覆盖。
- OS / 出口对白分离，OS 全员闭口。
- 多人对白强制正反打。
- 说话人用画面方位 + 外貌锚点指认。
- 六宫格/五宫格硬切可用黑屏冻结锚降低整图一起动的风险。
- 相邻子格景别与机位跳变，但世界连续、不越轴。
- 角色、场景、道具必须先提取并锁定，再生成分镜。
- 视频提示词必须包含场景音效、拟音、道具音、动作音和音乐建议。

## implementation_policy 实现策略

- 只提炼规则，不复制原模板正文。
- 把规则落到数据字段、校验规则、返修逻辑，而不是做提示词堆叠。
- v0.2 先独立为 `v02_*` 模块，稳定后再合并到主 CLI。
